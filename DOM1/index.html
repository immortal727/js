Функцию generateField(n) по генерации игрового поля размером n x n. Значение n не может быть меньше 3. Для 3х ячеек поля (для выбора конкретной ячейки использовать random) добавить атрибут prise. Значения атрибута prise устанавливаются из массива. Для 1й ячейки значение атрибута prise="cat", для 2й ячейки значение атрибута prise="book", для 3й ячейки значение атрибута prise="car"
